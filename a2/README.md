# How to Run

## Start the Server

In **a terminal**, run:

python3 mychatserver.py

In **seperate terminals**, run multiple clients:

python3 mychatclient.py

## Chat Format

Messages from other clients will appear in this format:
<port_number>: <message>

to leave the chat, type *exit*
